JetBrain 툴에서는 Live Template 라고 한다.

Preferences>Live Templates 항목에서 편집 가능하고

또는 코드를 선택하고 Save As Live Template 메뉴를 띄워 등록한다.

타이핑시 자동완성으로 보여지는 템플릿을 바로 선택해서 사용할 수 있다.

매개변수와 기본값을 지정하여 사용할 수 있다.
