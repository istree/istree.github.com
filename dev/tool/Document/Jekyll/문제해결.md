## You have already activated ..., but your Gemfile requires ... Prepending `bundle exec` to your command may solve this.

bundle exec로 실행한다.

```bash
bundle exec jekyll serve
```

## Liquid Exception: No repo name found.

_config.yml에 다음을 추가

```bash
repository: "username/repo-name"
```
